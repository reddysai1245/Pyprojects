to Run this follow the below steps

requirements
cv2
numpy
pandas
argparse
time

Open command prompt and go to the path where you downloaded this folder
and type
python "color_detection (1).py" -i IMAGE

your image will be displayed 
you can pick your desired color from that image 

If you want to change the image follow the below
delete the current image in the folder
rename your image with "image.jpg"
